# Zeolite Regex Example

*Also see
[a highlighted version of the example code](https://ta0kira.github.io/zeolite/example/regex/index.html).*

To run the example:

```shell
# Compile the example.
../../zeolite -i ../../lib/util -m RegexDemo .

# Run the unit tests.
../../zeolite -t .

# Execute the compiled binary.
./RegexDemo
```
